# Sign Up Readme

## 简介

谢谢您的使用~这是一个简单的注册系统实现~他在搭载Chrome的Windows10平台下有着不错的体验，希望您能喜欢。

## 如何使用？

在/bin目录下执行

```shell
node server.js
```

然后，通过Chrome浏览器访问127.0.0.1:8888即可开始您的体验。

## 亮点

1. 好看（大概吧
1. 通过AJAX实现的不重新加载的情况下的前后端交互

## 推荐配置

1. OS: Windows 10
1. Browser: Chrome
1. resolution radio: >= 1080p
1. runtime: node v9.2.0+
